# 西安旅游路线图 MVP

这是一个纯静态页面，不需要打包构建，适合手机、微信内置浏览器和普通浏览器打开。

## 本地预览

```bash
python3 -m http.server 8099
```

然后打开：

```text
http://localhost:8099/
```

## 部署

把以下文件放到任意静态托管目录即可：

- `index.html`
- `styles.css`
- `app.js`

可选部署方式：

- GitHub Pages：把项目推到仓库，Pages 选择根目录发布。
- Nginx：把三个文件复制到站点根目录。
- 宝塔/静态网站托管：上传三个文件即可。

## 功能

- 按 `西安旅游攻略.xlsx` 的 5 天路线生成日程卡片。
- 每个目的地支持高德、苹果地图、腾讯地图搜索/导航入口。
- 每天提供餐厅或小吃关键词，支持直接打开大众点评/美团搜索。
- 无后端、无数据库、无依赖，方便转发给同行人。
